# Legacy Marketplace Layer

## Purpose

This document explains the remaining marketplace-related components inside AutoConnect.

AutoConnect originally started as a service marketplace concept connecting customers and providers.

The platform direction later evolved into a Connected Vehicle Intelligence Platform.

## Current Status

Marketplace components are considered legacy and secondary.

They may later be used only as a service fulfillment layer after the vehicle intelligence engine recommends a maintenance or repair action.

## Legacy Components

- customers
- providers
- services
- bookings
- reviews
- matching_recommendations

## Future Decision

These components should either be:

1. Removed completely
2. Moved into a separate service marketplace module
3. Kept only as a service fulfillment layer

## Current Product Focus

The main product focus is now:

- vehicle telemetry
- diagnostics
- vehicle health
- EV battery intelligence
- hybrid efficiency
- driver behavior
- maintenance prediction
- insurance risk
- vehicle valuation
- ownership intelligence
